Prosjektbeskrivelse: 
Websiden er laget for et internasjonalt firma som driver med supply, trading, porteføljeforvaltning innenfor energibransjen.

Utviklere:
Bjarne Johansen

Web design: 
Nettstedet ble designet etter progressive enhancement basert på siden fra første inneleveringsoppgaven. Denne ble utvidet til å støtte ulike typer skjermoppløsning via css og en slider basert på javascript ble lagt til for desktop brukere. 

Javascriptet følger med på hvilken skjermoppløsning brukeren har og skjuler/viser slideren på første side baser på dette.

Teknologi brukt for implementasjon:
Javascript
CSS3
HTML5
